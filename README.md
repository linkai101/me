## 🛠️ Installation & Set Up
1. Install dependencies
```bash
npm install
```
2. Start the development server
```bash
npm run dev
```

## 🚀 Deploy
To deploy a static site:
1. Go to `package.json` and change the build script to `next build && next export`
2. Run the build script
```bash
npm run build
```
